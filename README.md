# Smart Message Analyzer 🤖

## 📌 Overview
This project is an enhanced version of a spam detection system. It not only detects spam messages but also provides study-related suggestions based on user input.

## 🚀 Features
- Detect spam messages
- Provide study tips
- Analyze normal messages
- Simple web interface

## 🛠️ Tech Stack
- Python (Flask)
- HTML, CSS

## ▶️ How It Works
The system checks user input against predefined keywords:
- Spam keywords → spam alert
- Study keywords → study suggestion
- Others → normal message

- Add machine learning model
